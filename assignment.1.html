How does this game work and how is it played?

Geomatry dash is a game about precision and timing. The concept of the game is to beat levels, and the more you progress, the harder the levels become. The game has been out for years, and it is still decently popular. The game is based on one button; that's it. only one button, but don't let that foul you because although the game has the minimum amount of controles a game can have, it is still considered one of the hardest.
